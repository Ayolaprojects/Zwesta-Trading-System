# Cuisine Express Mobile Depot Website

Corporate marketing website for a fuel supply, petroleum trading, and logistics business, built with React, TypeScript, and Vite.

## What is included

- Responsive landing page with an energy-sector visual direction.
- Local SVG art assets for hero, logistics, storage, and trading sections.
- Sections based on the supplied business profile: hero, vision, mission, services, industries, advantage, growth, and contact.
- Contact placeholders where the provided document did not include final email and phone details.

## Run locally

```bash
npm install
npm run dev
```

## Build for production

```bash
npm run build
```

## Content follow-up

Replace the email and phone placeholders with the final approved contact details and swap the SVG illustrations for official company assets if available.
